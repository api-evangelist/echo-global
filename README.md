# Echo Global Logistics

Tech-enabled freight brokerage with REST APIs for shipping quotes, booking LTL and FTL shipments, tracking deliveries, and managing carrier relationships.

## EchoSync API Suite

Echo Global Logistics exposes their platform capabilities through EchoSync, a suite of REST APIs available to partners, customers, and carriers:

- **Partner-Connect API** — Load creation (Truckload, LTL, Partial), LTL freight rating, real-time tracking, and document retrieval
- **Customer API** — Competitive truckload rate quoting 24/7/365
- **Carrier API** — Digital Freight Marketplace access with load visibility, offer placement, and Book It Now functionality
- **Authorizer API** — OAuth 2.0 Bearer token management for all APIs

All APIs use OAuth 2.0 Bearer token authentication. Access is partner-based and requires an existing relationship with Echo Global Logistics.
